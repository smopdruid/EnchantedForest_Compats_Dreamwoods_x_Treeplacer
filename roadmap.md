# Dreamwoods x Treeplacer Compat - Roadmap

## Project Overview
Compatibility datapack for allowing Dreamwoods' custom trees to be grown from sapling via Treeplacer.

## Goals & Scope
- [ ] Tree placement integration
- [ ] Tree facelifts using DW models as base
- [ ] Plant features consolidation

## Feature List
* Treeplacer support for Dreamwoods'trees
* Tree facelifts and expansions
* Possible biome additions

### Core Features
- [ ] Access to Dreamwoods' custom trees in Survival Mode through regular sapling growth.
- [ ] Biome-based weights reflecting climate compatibility. Large models grow better in ideal climates, shrubs dominate in poor climates, special trees show up more often in their native biomes (i.e. swamp trees in swamp biomes)
- [ ] Extra tree models added where appropriate

## Technical Notes
- Things to consider: Huge number of tree models, some are also quite large. Lots of trees with mismatched log/leaf combos. Sapling will always be tied to the leaf choice (acacia leaf + oak log = acacia sapling)
- Dependencies: Dreamwoods, Hexalia, Patchouli, Treeplacer (optional)
- Conflicts to resolve: Willow duplicate (resolved); Lavender duplicate
